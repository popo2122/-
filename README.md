# 풀무원 베이비밀 사이트 리뉴얼
> 사이트에 사용된 모든 사진은 상업적 목적이 아닌 개인 포트폴리오 제작용으로 사용되었습니다.      

## *💛 작업툴*
- 와이어 프레이밍 및 디자인 : Photoshop, Illustrator
- Bracket, Visual Studio code, jQuery    

## *💛 Design*
- 처음으로 만든 포트폴리오 사이트
- 베이비핑크와 베이지색을 이용하여 아기들이 먹는 '이유식'이라는 주제에 맞는 따뜻한 느낌을 주고자 함
- 오렌지 색은 브랜드 고유의 색으로, 강조 및 아이덴티티를 살리기 위해 중간중간 포인트 색으로 사용.
- 메인 슬라이드에서 좀 더 효과적으로 연령별 알맞은 제품을 알리기 위해, 사용된 재료의 사진과 실제 판매하는 이유식 이미지 배치
  > - 텍스트를 왼쪽으로, 이미지를 오른쪽으로 배치하여 가독성 고려
- 폰트는 깔끔한 가독성을 위해 'Noto sans'를 사용
- 사용자가 최대한 쉽게 제품을 알아보고, 구매에 접근할 수 있도록 메뉴와 fixed버튼 구성

  
## *💛 Code*
- PC~MOBILE버전까지 반응형을 지원하며, PC는 최대 1920＊1080 에서 최적화
- 메인을 포함한 각 섹션의 슬라이드는 jQuery로 구현
  > - addClass, removeClass사용하여 버튼 애니메이션 구현
  > - Click수는 변수를 사용하여, 인식하도록 만듦
- 공지사항 및 이벤트 부분 & 모바일의 메인 이미지 슬라이드는 'Swiper' 라이브러리를 사용
  > - 참고 링크 : [Swiper](https://swiperjs.com/)
- fixed button인 +버튼을 클릭했을 때, 사용자가 필요한 정보가 자연스럽게 나오도록 jQuery사용하여 구현
